# ASOS

This app is a replica of the home screen in the ASOS app with some changes. The app supports Dark Mode and Dynamic Type for both iPhone and iPad. The iPad version uses a Sidebar instead of a Tab Bar. The below images demonstrate these features. The original document with these images can be seen at: https://docs.google.com/presentation/d/1vXCGDnZ7N-c4jeE8uTYTTLHTHU9ETIQMv9QTVNE3p-U/edit?usp=sharing
